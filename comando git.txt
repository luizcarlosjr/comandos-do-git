comandos git

suas configuracao do git

git config --global user.name "nome do github/gitlab"
git config --global user.email nome do email cadastrado no github/gitlab

comandos git 

git init ---> inicializar o repositorio no seu pc ou notebook
git clone ---> clonar o seu repositorio
git checkout -b nome da branch ----> criar uma branch
git checkout nome da branch ----> entrar na branch
git branch ---> ver as branch criada do repositorio
git add . ---> adicionar todos os arquivos da pasta no repositorio
git add nome do arquivo ---> adicionar arquivo especifico no repositorio
git status ---> ver como esta o status do repositorio
git commit -m "comentario" ---> comentar o arquivo adicionado
git push origin nome da branch ---> subir para o repositorio
